Easy Contact Chat displays a WhatsApp chat in your web. The perfect solution for your ecommerce or blog. 


== DESCRIPTION ==

Easy Contact Chat provides you with an easy way to communicate with your users. This plugin displays an icon or button in your web or ecommerce. When the user clicks the button the plugin opens a new WhatsApp chat.

== EASY SETUP ==
Just introduce your phone number adding the country dial-in code. You can configure features such as style, texts and animation in the admin panel.


== FEATURES ==

* Fully customizable: modify size, position and style
* 3 cool animation effects
* Compatible with Woocommerce
* Choose between 21 icon styles or use your custom icon
* Shortcode compatible. Put a contact button wherever you want.
* Compatible with Google Analytics. You do not need to configure anything.
* Responsive
* Available languages: English, Spanish
* Translation support
